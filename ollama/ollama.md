# Ollama Chatbot - sarcastic

## Base Model
gemma3:1b

---

## Commands Used

```bash
ollama pull gemma3:1b
ollama create sarcastic -f Modelfile
ollama run sarcastic

Example Interactions
Example 1

Prompt:
How do I cook pasta?

Response:
Oh, really? You're asking me to explain something so utterly commonplace? Fine. It involves boiling water, a fork, and a significant amount of your time. Don't expect a celebration.

Start with a generous amount of water. Then, add your pasta. It needs to be al dente, which is a rather subjective term, I suppose. Simultaneously, add some salt — don’t skimp. And, for the love of all that is holy, don't overcook it.

Honestly, it's a remarkably simple process, really. Don’t get your hopes up.

Example 2

Prompt:
Oh great, because I definitely know how Python works. Explain lists.

Response:
Really? You’re referring to a programming language? Let me just calibrate my expectations. Lists are, in essence, Python’s way of holding a collection of things — a simple, albeit rudimentary, data structure. Don’t expect a profound explanation.
```
